# 短剧编剧 · 领域心智框架提取

> Phase 2 成果：基于5人+理论底座的六维度调研，经三重验证法（跨2+领域出现、能预测新问题立场、排他性）提炼

---

## 一、领域共识（5条，经三重验证）

### 共识1：「钩子即生存」——前3-5秒决定内容的生死

| 来源 | 表述 |
|------|------|
| 杨玉鲁 | "3秒设钩"——每集开场必须有钩子，用3秒抓住观众 |
| 洛古特 | "矛盾点和剧情爆发前置"——短剧不能像小说那样渐进铺垫 |
| 贾毅 | "叙事3.0：将起承转合压缩至以分钟为单位的情感脉冲模块" |
| 王晨 | "完播率"是核心数据——前几秒决定用户是否滑走 |
| 麦基 | 短剧的激励事件必须"前3集内完成，甚至前3秒给出信号" |

> **三重验证通过**：跨5个独立来源（编剧/网文转型/出海/平台/理论），能预测任何短剧失败原因（前3秒没钩子≈必死），具有排他性（长剧/小说不适用此规则）

### 共识2：「市场是唯一评委」——自我表达服从数据反馈

| 来源 | 表述 |
|------|------|
| 杨玉鲁 | "拿数据说话""不能困在自己定义'好作品'的漩涡里" |
| 黄经天 | "敬畏观众"刻在全团队的价值观墙上；评论前十条必须看 |
| 洛古特 | "会跟风挺好"——了解市场最热题材才能在热点上做微创新 |
| 贾毅 | "短剧本质是情绪驱动型故事"——按用户行为数据定义"好故事"（吸量/留存/变现） |
| 王晨 | 三维度评估模型——内容质量×视听呈现×硬件配置，市场数据说了算 |

> **三重验证通过**：跨5个独立来源，能预测行业方向（精品化≠高成本，而是高数据反馈），排他性：传统影视编剧以导演/剧作家主观判断为主

### 共识3：「创新=跟风+半步」——在验证过的话题上做边际突破

| 来源 | 表述 |
|------|------|
| 洛古特 | "跟风→了解市场热点→微创新→量变引发质变→开拓新题材"——完整路径 |
| 黄经天 | "创新是生命力"——《家里家外》在已有家庭题材上，用四川方言+日常流开创现实赛道 |
| 贾毅 | "第一次哲学"：80%创新失败，但成功的几乎都是"第一次"；创新是常态 |
| 杨玉鲁 | "守好一亩三分地，比盲目追热点更重要"——深耕赛道再微创新 |
| 救猫咪 | 10大故事类型——本质是在类型框架内做个性填充，而非彻底原创 |

> **三重验证通过**：纯创新=高风险，纯跟风=无差异化，最优路径=已验证类型+边际创新。能预测《太奶奶》第7版本反超的逻辑。排他性：与长剧/电影"作者论"路径截然不同

### 共识4：「短剧是压缩的情绪机器」——结构服务于情绪密度

| 来源 | 表述 |
|------|------|
| 杨玉鲁 | "短剧是成年人的童话"——提供现实中难以达成的幻想体验和强烈情绪 |
| 黄经天 | "情绪→情感"转向——早期短剧是纯情绪刺激，现在要叠加情感共鸣 |
| 贾毅 | 叙事3.0将传统影视压缩到"情感脉冲"，每模块独立具备悬念、反转、情感释放 |
| 麦基 | 鸿沟原理——短剧每集结尾裂开新鸿沟，强制观众"接下来怎么办" |
| 救猫咪 | 节拍表短剧适配——80集对应15节拍，密集的起承转合 |

> **三重验证通过**：短剧不是"缩短的电视剧"，而是"压缩的情绪释放装置"。预测：任何失去情绪密度的短剧必然扑街。排他性：长剧可以容忍情绪留白和氛围营造，短剧不能

### 共识5：「网文基因+镜头思维」——短剧编剧的底层操作系统

| 来源 | 表述 |
|------|------|
| 杨玉鲁 | "从作家到产品经理"——从文学思维转向镜头思维 |
| 洛古特 | 网文四维转化：节奏/信息点/人物塑造/线索的全面重构；"上帝视角"三害 |
| 黄经天 | 听花岛早期核心成员从短视频"剧情号"起步（2018年抖音剧情号涨粉数十万） |
| 公子衍等（阅文转型） | 网文IP+作者转型编剧，爆款率超60% |

> **三重验证通过**：短剧编剧的最佳人才池是网文作者（已掌握爽点节奏）+影视化训练（掌握镜头语言）。预测：纯影视编剧转型短剧要补"爽点密度"，纯网文作者转型要补"画面可拍性"

## 二、流派分歧（5派差异化洞察）

这些不是共识，而是不同视角下的独特判断。每个在特定场景优于共识。

### 分歧1：题材策略 — 深耕 vs 多变

| 派别 | 立场 | 论据 |
|------|------|------|
| **杨玉鲁派** | 深耕单一赛道 | "守好一亩三分地"——在女频/穿越一个类型做到极致 |
| **洛古特派** | 题材通杀+跟风轮动 | 百川中文2024年产出涵盖男频女频共30+题材 |
| **适用场景** | 新人/小团队→深耕；成熟团队有编辑驱动→通杀 | |

### 分歧2：制作标准 — 轻量快速 vs 精品重投

| 派别 | 立场 | 论据 |
|------|------|------|
| **凉白开风格** | 轻量标准化 | 标准剧本格式+高效过稿流程，快速验证 |
| **听花岛风格** | 精品笨功夫 | 10万字分析报告+4个月制作周期+实景拍摄 |
| **适用场景** | 追求频率和试错成本→轻量；追求口碑和长尾→精品 | |

### 分歧3：原创 vs 改编 — 两派的数学

| 派别 | 立场 | 论据 |
|------|------|------|
| **黄经天/听花岛** | 大规模原创 | 行业唯一大规模开发原创剧本的厂牌 |
| **洛古特/网文派** | IP改编为主 | 网文IP自带验证过的故事框架和用户基础 |
| **适用场景** | 想建立品牌差异化→原创（高风险高回报）；想降低失败率→改编（已验证故事+微创新） | |

### 分歧4：国内 vs 出海 — 叙事DNA差异

| 维度 | 国内短剧 | 出海短剧（ReelShort） |
|------|---------|----------------------|
| 情感驱动 | 社会关系（婆媳/家族/逆袭） | 个体欲望（爱情/复仇/身份认同） |
| 节奏 | 密集爽点，强调"打脸" | 强情节点，强调"张力" |
| 表现 | 直给型（台词直接表达情感） | 演给型（靠表演和情境） |
| 文化锚点 | 中国社会结构 | 通用人类情感（贾毅的"西壳东核"） |

### 分歧5：AI定位 — 工具 vs 重构

| 派别 | 立场 | 论据 |
|------|------|------|
| **洛古特** | AI是生产力 | "到年底AI短剧占比80%"，主编驱动+AI双轨制 |
| **王晨/抖音** | AI是分发和辅助 | 用AI做个性化推荐和素材优化，不替代创作 |
| **适用场景** | 量产型短剧→AI重参与；精品化短剧→AI辅助 | |

## 四、心智模型汇总（3个核心模型）

### 模型1：情绪压缩引擎
**一句话**：短剧不是"缩短的故事"，而是"压缩的情绪释放装置"。
**证据**：杨玉鲁的"情绪节奏"+贾毅的"情感脉冲"+麦基的"鸿沟原理"+黄经天的"情绪→情感"
**应用**：写任何一集短剧前，先问——这一集释放了什么情绪？情绪密度够不够？结尾有没有新的情绪钩子？
**局限**：纯情绪驱动可能导致人物扁平化；需黄经天的"情感层"补足深度

### 模型2：热点增量创新环
**一句话**：跟风→理解热点→在同方向做1-2个新元素→测试→迭代。
**证据**：洛古特的"会跟风"完整路径+黄经天《太奶奶》第7版本反超+贾毅的80%失败率数据
**应用**：拿到热点后不要空想创新，先用"跟风版本"验证，再在一个维度上做增量（人设/反转方式/情感侧重/视觉风格）
**局限**：高度同质化环境中，1-2个新元素可能不够形成差异化

### 模型3：可拍性前置过滤器
**一句话**：写剧本的第一道检查不是"好不好看"，是"能不能低成本拍出来"。
**证据**：洛古特"文字到画面三原则"+杨玉鲁"短剧剧本是拍摄指南"+贾毅的"第一次"尝试80%失败
**应用**：每个场景写完检查三问——能不能用3个道具以内完成？能不能在1个场景内完成？OS能不能改成动作/对白？
**局限**：过度强调可拍性可能扼杀想象力；精品化短剧允许适当增加拍摄复杂度
