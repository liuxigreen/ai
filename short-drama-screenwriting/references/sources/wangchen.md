#### 抖音精品短剧扶持政策演进（2024-2026）

**2024年1月 - 辰星计划发布**:
- 联合出品模式：抖音以现金投资方式联合出品，提供全流程支持
- 联合运营模式：创作者负责开发制作，抖音提供千万级流量扶持
- 最高500万现金 + 1亿流量扶持

**2024年9月 - "首亮微光"2.0扶持计划**:
- 8家平台整合资源，一年内推出400个精品微短剧项目
- 分账模式、广告收益、流量扶持全面升级

**2025年5月19日 - 三大计划全面升级**:
- **辰星计划**（面向编剧和制作方）:
  - 新增精品定制模式：单部最高500万保底，平台100%承担制作费用
  - 投资出品模式升级：投资比例提至70%，制片方可额外享受50%利润分成
  - 典型作品：《大话大话西游》《夫妻的春节》《我在冷宫忙种田》
- **星火计划**（面向短剧机构）:
  - 单片保底和框架保底合作，单部最高投入500万元
  - 典型案例：《但行好事》4.1亿播放，超500万元累计收益
- **星光计划**（面向达人和MCN机构）:
  - 10万粉以上各类作者均可参与
  - 典型案例：达人"小雨老师"（968万粉）通过《觉醒的她》（1.9亿播放）强化人设，多部短剧累计收益超600万元
- **来源**: 知乎专栏（2025年5月19日）

**2025年8月 - 漫剧扶持政策**:
- 番茄IP漫剧改编计划：开放6万+部番茄小说网文原创IP
- 渠道分发：抖音、红果短剧等多渠道上线
- 收益分成：全渠道免付费净收益的90%分成给版权方
- 成片内容框架保底合作：月产能≥5部的版权方可签框架协议
- 头部精品漫剧单部采买合作：总时长不低于120分钟，单集1-3分钟
- **来源**: 腾讯新闻（2025年8月15日）/ 搜狐（2025年8月16日）

**2026年初 - 影视机构专场升级**:
- 新增影视IP合作专属赛道
- 单个IP最高可获80万元保底
- 单部成片激励最高达300万元
- **来源**: 腾讯新闻（2026年1月30日）

**2026年4月 - 真人短剧专项扶持政策**:

- **系列剧激励**: 季播剧、衍生剧、单元剧，续作上线越快、系列化程度越高，激励比例越高，最高20%额外分账上不封顶

## 维度3：表达DNA

### 3.1 抖音短剧平台核心行业术语

| 术语 | 含义 |
|------|------|
| **跑量** | 内容在平台上的实际分发和播放量表现，即"能跑多少量" |
| **完播率** | 用户完整看完一集的比例，竖屏短剧核心指标 |
| **竖屏思维** | 适配手机竖屏观看的叙事逻辑和构图方式 |
| **情绪价值** | 短剧提供的爽感、甜感、虐感等即时情绪满足 |
| **卡点/钩子** | 每集末尾设置的悬念或反转，驱动用户持续观看和付费 |
| **辰星计划** | 抖音精品短剧共创计划（2024.1启动） |
| **星火计划** | 面向短剧机构的扶持计划 |
| **星光计划** | 面向达人和MCN机构的扶持计划 |
| **精品化** | 从粗制滥造向影视级制作的品质升级方向 |
| **系列化/IP化** | 从单部剧到季播、衍生剧的长线运营模式 |
| **IAP/IAA** | 应用内购买（付费解锁）/ 应用内广告（免费观看）两种变现模式 |
| **原生短剧** | 在抖音站内直接创作、分发的短剧，区别于小程序短剧投流 |
| **微短剧+** | 微短剧与文旅、品牌、非遗等跨界融合模式 |
| **分层分类审核** | 国家广电总局对微短剧的管理制度 |
| **投流** | 通过广告投放获取用户、驱动播放量 |
| **分账** | 平台与版权方按比例分配收入 |
| **保底** | 平台对优质内容的保障性最低收入承诺 |
| **热力值** | 综合衡量短剧市场热度的评价指标 |

### 3.3 抖音短剧内容评估框架（王晨提出的三维度模型）

抖音短剧内容质量评估 = 内容质量 × 视听呈现 × 硬件配置

其中：
- **内容质量**：人物塑造 + 情绪价值 + 故事情节 + 差异化
- **视听呈现**：构图光影 + 配音配乐 + 后期剪辑 + 服化道突破
- **硬件配置**：题材 + 元素 + 主创人员 + 主要演员

## 维度4：外部视角

### 4.1 创作者对抖音短剧生态的评价

**正面评价**:
- 抖音政策覆盖创作全周期（剧本端→成片端→上线→续作），降低中小团队创作风险
- "一键托管"模式降低版权方运营成本，优质剧目整体收入提升2-3倍
- 2024年原生付费短剧账号年均涨粉达120万，复购率和复访用户黏性持续提升
- 星光计划为不同成长阶段达人提供差异化扶持

**行业观察**:
- 2025年下半场共识："内容为王"取代"流量红利"
- 行业从追求数量转向深耕质量，同质化内卷走向差异化精品化竞争
- 制作成本两极分化：20万以内基础质量剧 vs 80万以上精品剧
- 头部制作公司规模化效应明显，中小团队突围难度增加

### 4.2 抖音 vs 快手短剧差异化策略

| 维度 | 抖音 | 快手 |
|------|------|------|
| **短剧渗透率** | 66.1% | 73.6% |
| **商业模式** | 多元化（IAP + IAA + 免费 + 付费混合） | 付费会员模式（2.8-16.8元/月） |
| **核心优势** | 用户基数庞大、流量优势、算法分发强 | 短剧根基深厚、社群粘性高、直播协同 |
| **内容特色** | 搞笑剧情、影视娱乐类创作者受追捧 | 影视娱乐类账号占比超80% |
| **特色产品** | 红果短剧（免费模式，月活1.4亿，同比增长626%） | 喜番短剧（月活309万） |
| **策略重心** | 精品化 + IP化 + 全产业链投资（制片中心模式） | IAP/IAA + 付费会员三线并行，培养用户付费习惯 |
| **创作者扶持** | 三大计划分层覆盖 + 15亿保底预算 | 会员费全部让利合作方（扶持期免抽佣） |
| **关键判断** | 抖音正从"流量场"升级为"制片中心" | 快手深耕短剧社群的付费转化 |

- **来源**: 搜狐（2024年12月17日/24日）、DataEye/36氪（2025年12月）

### 4.3 行业报告中的短剧市场数据

**市场规模**:
- 2024年中国微短剧市场规模505亿元，首次超过电影票房
- 2025年免费微短剧市场规模约533亿元，同比增长113%
- 2025年微短剧总产值预估约1000亿元（含免费+付费）
- 预计2027年达到856.5亿元，年复合增长率19.2%

**用户规模**:
- 截至2024年底用户规模6.62亿，占网民近六成
- 下沉市场与中青年群体成主力军
- 红果短剧2024年11月月活1.4亿（同比增长626%）

**内容供给**:
- 2024年抖音上播放量破亿的微短剧达500部，其中12部破10亿
- 2025年全年新剧约4万部创新高
- 漫剧2025年市场规模预计突破150亿，83%复合增长率
- 2026年3月：2部单月分账破800万，200+部破100万，10家版权方月分账总额破3000万

- **来源**: DataEye研究院、QuestMobile、艾媒咨询、36氪

### 4.4 行业趋势判断

**2025年下半场 vs 上半场核心差异**:
- 上半场：流量红利、快速复制、以"有一个出彩点即可立足"
- 下半场：内容为王、精品化深耕、"无短板比拼"
- 付费大盘下滑（11月同比降7%），免费模式占比持续提升
- 漫剧成新增长极，AI技术赋能降本增效（单集成本降至500元）

## 维度5：关键决策与案例

### 5.1 抖音短剧品类规划与资源倾斜

**核心品类矩阵**（据多源整合）:

| 品类 | 定位 | 趋势判断 |
|------|------|----------|
| 都市逆袭（男频） | 主流基本盘 | 播放量与互动量显著高于均值 |
| 女性重生/复仇（女频顶流） | 核心变现品类 | "逆袭"标签互动量高出均值40% |
| 甜宠恋爱 | 全受众通吃 | 横屏版追求质感与情感细腻度 |
| 古装权谋/仙侠 | 古风大热门 | 服化道与特效升级，向正剧质感靠拢 |
| 年代温情/家庭伦理 | 全家欢爆款 | 80-90年代烟火气引发集体怀旧 |
| 现实主义 | 社会议题爆款 | "暴击+救赎"路线，话题度拉满 |
| 悬疑/谍战 | 强剧情品类 | 单集完播率高，民国+现代双线发力 |
| 非遗/文化 | 口碑+流量新蓝海 | 政策加持，小众题材大格局 |
| 文旅/地域 | 政策+流量双驱动 | 线下消费转化潜力大 |
| **漫剧** | **最新增长极** | 男频/古装为主，女频供给缺口大 |
| 系列化/IP剧 | 核心技术方向 | 平台最高20%额外分账上不封顶 |

**品类趋势判断**:
- **在涨的类型**: 年代温情、现实主义、非遗文化、漫剧、女性成长、亲情向
- **基本盘**: 都市逆袭、甜宠、古装权谋

### 6.3 平台政策演变脉络

```
阶段一：流量红利期（2023年前）
    - 野蛮生长，内容质量参差不齐
    - 依靠信息流和低门槛内容起步
    - 抖音以短视频分发优势自然获取短剧流量

阶段二：精品化起步期（2024年）
    - 辰星计划建立精品标准
    - 内容向"讲述真实生活、弘扬传统文化、融合文旅"转型
    - 联合出品+联合运营模式确立
    - 付费用户和金额爆发式增长

阶段三：版权中心整合期（2025年）
    - 版权中心成立，统一战略
    - 三大计划分层覆盖全产业链
    - 漫剧作为新增长极被重点扶持
    - 免费模式超越付费模式成为主流
    - 保底模式大规模引入降低创作风险

阶段四：生态重塑期（2026年）
    - 从"流量分发"升级为"制片中心"模式
    - 15亿保底预算规模化投入
    - IP化+系列化成为核心方向
    - 全渠道资源整合（抖音+红果）
    - 行业泡沫出清，优胜劣汰加速
```
